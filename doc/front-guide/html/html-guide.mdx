HTML作为描述网页结构的超文本标记语言，也需要统一代码风格。

## 缩进与换行

可以使用 4 个空格作为一个缩进，看大家习惯定义

每行不超过 120 个字符

## 标签

**标签名必须使用小写字母**。

```html
<!-- 推荐 -->
<p>Hello Style Guide!</p>

<!-- 不推荐 -->
<P>Hello Style Guide!</P>
```

**对于无需自闭合的标签，推荐自闭合**。

常见无需自闭合标签有 `input`、`br`、`img`、`hr` 等。

**对 HTML5 中允许省略的闭合标签，不允许省略闭合标签**。

```html
<hr/>
<!-- 推荐 -->
<ul>
  <li>first</li>
  <li>second</li>
</ul>

<!-- 不推荐 -->
<ul>
  <li>first
  <li>second
</ul>
```

**标签使用必须符合标签嵌套规则**。

比如 `div` 不得置于 `p` 中，`tbody` 必须置于 `table` 中。

**HTML 标签的使用应该遵循标签的语义**。

下面是常见标签语义

- `p` - 段落
- `h1`，`h2`，`h3`，`h4`，`h5`，`h6` - 层级标题
- `strong`，`em` - 强调
- `ins` - 插入
- `del` - 删除
- `abbr` - 缩写
- `code` - 代码标识
- `cite` - 引述来源作品的标题
- `q` - 引用
- `blockquote` - 一段或长篇引用
- `ul` - 无序列表
- `ol` - 有序列表
- `dl`，`dt`，`dd` - 定义列表

**标签的使用应尽量简洁，减少不必要的标签**。

## 属性

**属性名必须使用小写字母**。

```html
<!-- 推荐 -->
<td colspan="3">...</table>

<!-- 不推荐 -->
<td colSpan="3">...</td>
```

**属性值必须用双引号包围**。

不允许使用单引号，不允许不使用引号。

```html
<!-- 推荐 -->
<script src="esl.js"></script>

<!-- 不推荐 -->
<script src='esl.js'></script>
<script src=esl.js></script>
```

**自定义属性建议以`data-`为前缀**。

使用 `data-` 可以在 `dom` 中通过 `dataset` 属性直接获取值。

```html
<ol data-ui-type="Select"></ol>
```

## CSS 和 Javascript 引入

**引入 CSS 时必须指明 `rel="stylesheet"`**。

```html
<link rel="stylesheet" src="index.css">
```

**引入 CSS 和 Javascript 时无需指明 `type` 属性**。

**建议在 head 中引入需要的所有 CSS 资源**。

**建议 JS 脚本放在页面末尾，或采用异步加载**。

## 图片

**禁止 `img` 的 `src` 值为空**。

src 值为空，会导致部分浏览器重新加载一次当前页面，[https://developer.yahoo.com/performance/rules.html#emptysrc](https://developer.yahoo.com/performance/rules.html#emptysrc)

**建议为图片都加上 `alt` 属性**。

在图片加载失败时，会显示 `alt` 值。提高用户体验。

**建议添加 `width` 和 `height` 属性，以免页面抖动**。

## 按钮

**使用 `button` 元素时必须指明 `type` 属性值**。

**尽量不要使用按钮类元素的 `name` 属性**。

由于浏览器的兼容性问题，使用按钮的 name 属性会带来许多难以发现的问题，参考[此文](http://w3help.org/zh-cn/causes/CM2001)
